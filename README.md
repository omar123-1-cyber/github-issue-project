    - 1️⃣ What is the difference between var, let, and const?
    Difference between var, let, and const:

    var is function-scoped, redeclarable, hoisted

    let is block-scoped, not redeclarable, mutable

    const is block-scoped, not redeclarable, immutable
    - 2️⃣ What is the spread operator (...)?
    Spread operator (...)

    Array ba object elements ke expand korte use hoi,
    example:::
    let arr = [1,2]; 
    let arr2 = [...arr,3]; // [1,2,3]
    - 3️⃣ What is the difference between map(), filter(), and forEach()?
       Difference between map(), filter(), and forEach():

     map() - returns new array with modified elements.

     filter() - returns new array with elements that pass condition.

     forEach() - iterates but returns nothing.
